# EDA Section 06: SATCAT Analysis

## Goal
Generate first-pass descriptive insights from SATCAT relevant to debris-risk context.

## Analyses
- Object composition using `OBJECT_TYPE` frequency distribution.
- Launch activity trend over time using `LAUNCH_DATE` year extraction and yearly counts.

## Interpretation Focus
- Relative scale of payload, rocket body, and debris-like object classes.
- Growth dynamics in orbital object population over time.

## Relevance to OpenEnv
- Supports realistic environment initialization and scenario priors.
- Helps define class-weighted sampling for simulation episodes.
